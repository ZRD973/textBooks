
## Project Description

```
项目前后端分离  web前端，server后端；
数据库创建textbooks，导入数据库；
在后端server配置数据库，server/config/config.default.js
分别运行项目，npm install        npm run serve
```




系统用户统一由 管理员 添加；管理员拥有所有权限；


## 首页 

#### 显示老师教的对应课程和班级，可以添加课程，删除课程

## 系统管理

#### 用户管理：对用户增删改查

#### 角色管理：管理角色，权限；

#### 菜单管理：管理菜单

## 教材仓库

#### 教材数据：显示所有教材

#### 教材征订  ：教师申请教材，图书馆管理员审核。审核通过——>加入仓库

#### 出入库记录：显示各班级领取书的记录，以及教材入库记录；

## 领书管理

#### 领书操作：班级负责人领取书

#### 领书分析：//占时没有

## 教务管理

#### 班级管理：显示课程信息，增删改查

#### 课程信息：显示课程信息，增删改查

#### 教师信息：显示所有的教师，查
